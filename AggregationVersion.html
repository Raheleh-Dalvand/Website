Aggregation Functions

AVERAGE/AVERAGEA/AVERAGEX Function

Calculates averages: AVERAGE for numbers only, AVERAGEA includes all values, and AVERAGEX averages an expression over a table.

COUNT/COUNTA/COUNTX/COUNTBLANK/COUNTROWS Function

Counts values: COUNT counts numbers, COUNTA counts all non-blanks, COUNTX counts expression results, COUNTBLANK counts blanks, and COUNTROWS counts table rows.

DISTINCTCOUNT/DISTINCTCOUNTNOBLANK Function

Counts unique values: DISTINCTCOUNT counts all distinct values, DISTINCTCOUNTNOBLANK excludes blanks.
